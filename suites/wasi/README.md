# Labs Conformance Tests

![labs](https://img.shields.io/badge/OJS-Labs-blueviolet)

These conformance tests cover **OJS Labs** features that are not yet part of
the core release train. They are included in the conformance suite for early
validation but are NOT required for core conformance certification (L0–L4).

A backend that passes L0–L4 core tests is fully conformant regardless of
whether it implements these Labs extensions.
